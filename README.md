Hunger Free
Hunger Free is a web application aimed at raising awareness about hunger and promoting social change towards a hunger-free world. The application provides users with the ability to sign up, creating an account and accessing exclusive content.

Features
Sign up functionality: Users can create an account, providing their email and password.
Awareness: The application presents information about hunger and its causes, raising awareness about the issue and encouraging users to take action.

Technologies
React: A JavaScript library for building user interfaces.
Firebase: A platform for developing web and mobile applications, providing authentication and database services.
How to run the application
To run the application on your machine, you need to:
Tailwind css
Clone the repository: git clone https://github.com/your-username/hunger-free.git
Install the dependencies: cd hunger-free && npm install
Start the application: npm start
Open your browser and navigate to http://localhost:3000
